# Update Frontend SDK (Incremental)

Incrementally update TypeScript SDK based on git changes.

## Target

- Generate in current project: `generated/sdk/`
- Copy to frontend: `/Users/easayliu/Documents/GitHub/orris-frontend/src/api/`

## Instructions

1. Run `git diff --name-only HEAD~1` or `git status --porcelain` to detect changed files

2. Filter for relevant changes:
   - `internal/interfaces/http/handlers/*.go`
   - `internal/interfaces/http/handlers/**/*.go`
   - `internal/application/*/dto/*.go`
   - `internal/interfaces/http/router.go`
   - `internal/interfaces/http/routes/*.go`

3. Map changed files to modules:
   - `authhandler.go` → auth
   - `userhandler.go` → user
   - `subscriptionhandler.go`, `subscriptionplanhandler.go`, `subscriptiontokenhandler.go` → subscription
   - `notificationhandler.go` → notification
   - `paymenthandler.go` → payment
   - `profilehandler.go` → profile
   - `handlers/node/*` → node
   - `handlers/forward/*` → forward
   - `handlers/ticket/*` → ticket
   - `handlers/admin/*` → admin
   - `application/*/dto/*` → corresponding module

4. Read the changed handler/dto files

5. Generate updated SDK files in `generated/sdk/{module}/`:
   - `types.ts`
   - `client.ts`
   - `index.ts`

6. Keep the file header:
```typescript
/**
 * AUTO-GENERATED - DO NOT EDIT
 * Generated by Claude from backend Go code
 * Source: orris backend
 * Last updated: {current date}
 */
```

7. Copy updated modules to frontend:
   ```bash
   cp -r generated/sdk/{module}/* /Users/easayliu/Documents/GitHub/orris-frontend/src/api/{module}/
   rm -rf generated/sdk
   ```

## Output

Report which modules were updated:
```
Updated SDK for modules: [list of modules]
Changed files detected: [list of files]
```

If no relevant changes detected, report:
```
No API changes detected. SDK is up to date.
```
